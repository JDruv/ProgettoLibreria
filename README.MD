## PROGETTO LIBRERIA DIGITALE 

ISTRUZIONI PER AVVIARE IL PROGETTO 
PER AVVIARE IL PROGETTO CREARE UN DATABASE CHIAMATO 'Libreria' SU MONGODB, SUCCESSIVAMENTE CREARE 3 COLLECTION : 
Utenti ,  Prestiti e Libri, ed importare i tre file nella cartella 'DatabaseMongoDB' 

APRIRE VISUAL STUDIO E APRIRE LA CARTELLA 'ProgettoLibreria' , APRIRE DUE TERMINALI DA VISUAL STUDIO 
SU UNO ANDARE NELLA CARTELLA CLIENT E SULL'ALTRO LA CARTELLA SERVER ED ESEGUIRE SU ENTRAMBI IL COMANDO 'npm install' 
INFINE DOPO L'INSTALLAZIONE DELLE LIBRERIE ESEGUIRE SU ENTRAMBI IL COMANDO 'npm run dev' 
ORA PER APRIRE IL SITO, APRIRE IL LINK DEL SERVER IN 'Localhost:5173'
